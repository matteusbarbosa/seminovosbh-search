## crawler-seminovosbh

executar composer install

execute o servidor php:
- cd (caminho do projeto)
- php -S localhost:8000 -t public

## Exemplos de uso

Listar carros : /?marca=Renault&pagina=1&cidade=2700

Mostrar Detalhes: /veiculo/{id}, EX = /veiculo/2589837
